Titanic Survival Model

We will be using Titanic DataSet for Analysing the Data and then finally predicting whether a passenger will going to be survived or not.

Data Overview

We are given data of passenger boarded on titanic

We have following columns

  1. PassengerID

    2. The id of the Passenger

  2. Pclass

    3. A proxy for socio-economic status (SES)
    1st = Upper
    2nd = Middle
    3rd = Lower

  3. Name

    4. The name of the passenger

  4. Sex

    5. The Sex of the passenger(Male/Female)

  5. Age

    6. It denotes the age of the Passenger

  6. SibSp

    7. The dataset defines family relations in this way
    Sibling = brother, sister, stepbrother, stepsister
    Spouse = husband, wife (mistresses and fiancés were ignored)
    This column tells us about number of Siblings/spouse that passenger have

  7. Parch

    8. The dataset defines family relations in this way…
    Parent = mother, father
    Child = daughter, son, stepdaughter, stepson
    Some children travelled only with a nanny, therefore parch=0 for them.

  8. Ticket

    9. Ticket number of the passenger

  9. Fare

    10. Fare of the journey for that particular passenger

  10. Cabin

    11. It defines cabin number of the passenger some are None as all the passengers were not alloted the cabin

  11. Embarked

    12. point of embarkation 1.Q=QueensTown
    2.S=Southampton
    3.C=Cherbourg

Data Analysis

We performed Data Analysis on above mentioned dataset to know the information about our data and to see what our data tells us.
We first checked whether our datatset contains null values or not?
We found that Age,Cabin and Embarked had some null values.
We will also check whether a particular columns have any relations with survival column or not?
From the data you can clearly see that columns like PassengerID,Name are not useful to us so we will simply drop these columns from our dataset.
After further analysis we found out that Ticket,cabin,Fare are also not useful for us.

Conclusions after Data Analysis

  1. Out of 891 passengers 342 survived
  2. The chances of survival of females are higher than that of males
  3. 3. The chances of survival of passengers on basis of class is like: 1st class>2nd class>3rd class 4. The chances of survival of children and Adult Females is more than that of Males and it decreases as the age pf the male increases. 5. The chances of a Family person surviving is greater than an alone passenger surviving.
